# Interview Kit Skill — Role-Scoped Interview Question Generator

> **Generate realistic interview questions with working reference solutions and calibrated rubrics.** Coding, system design, and take-home rounds — stack-agnostic, level-scaled from junior to staff+.

**Keywords**: interview questions generator, technical interview kit, coding interview, system design interview, take home interview, hiring rubric, ai interview prep, interview kit claude code skill

## Install

```bash
curl -sSL https://raw.githubusercontent.com/Sanmanchekar/skills-library/main/install.sh | bash -s -- interview-kit
```

## What it does

- **Interview mode** — asks role / level / stack / round type / signals BEFORE generating
- Difficulty scaled from **junior → staff+** with a level table
- Every question ships with a **working reference solution** — no unsolved questions
- Every question ships with a **rubric distinguishing signals**, not just "did they finish"
- **Time budgets**: coding 45-60 min, take-homes capped at 6h (respect candidate time)
- Blocks LeetCode-hard and puzzle questions unless deliberate
- Same question can serve multiple levels with a **higher rubric bar**, not a harder problem
